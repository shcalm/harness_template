* `[ ]` 待处理
* `[/]` 进行中
* `[x]` 已完成
* `[!]` 阻塞

## 5. 会话交接

### 5.1 会话心智摘要

> 受众：code agent（compact 或续会时首先阅读）。
> 形式：自由文本，≤10 行，覆盖式更新，不累积历史。
> 内容：当前心智模型、近期思路、被否决路径、待验证假设的背景。结构化字段（焦点 / 下一步 / 阻塞）走表格，不在此重复。

```text
[本会话当前的工作记忆：
 - 我正在解决的核心问题是 ...
 - 上次中断前的判断是 ...
 - 已被否决的路径及原因 ...
 - 当前最大的不确定性 ...]
```

### 5.2 接力字段

| 字段 | 内容 |
| :--- | :--- |
| 最近完成 | `[本轮已经完成的具体事项]` |
| 最近证据 | `[TEST-ID / 日志路径 / 命令]` |
| 恢复入口 | `[下一次打开项目后第一条命令或第一份文件]` |
| 风险 | `[未解决风险和对应 BUG/TASK-ID]` |
